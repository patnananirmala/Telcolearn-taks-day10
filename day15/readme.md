# Hands-on Examples — JSON, YAML, Telecom APIs & Router Configs

This repository contains simple hands-on tasks related to working with JSON, YAML, telecom-style API data, and router/network configuration files.  
All tasks are implemented using small and easy Python scripts.

---

## **Tasks Covered**

### **1. Load and Parse JSON Files**
- Read JSON files from sample telecom APIs.
- Pretty-print and validate JSON structure.

### **2. Convert YAML ↔ JSON**
- Convert YAML to JSON.
- Convert JSON back to YAML.
- Save both outputs into files.

### **3. Extract Nested Fields from API Responses**
- Work with nested telecom-style JSON.
- Retrieve values such as:
  - Device status  
  - Counters  
  - Statistics  
- Handle missing keys safely.

### **4. Build a JSON-Based Configuration Generator**
- Create JSON templates for router/network test setups.
- Generate configs based on:
  - Device name  
  - Interface IPs  
  - Netmasks  
  - Default routes
